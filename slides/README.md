# AI-CAST Demo Day Slides

Built with [Slidev](https://sli.dev).

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Export to PDF
npm run export
```

## Structure

- `slides.md` - Main presentation content
- `components/` - Vue components (if needed)
- `public/` - Static assets

## Demo Day Flow

1. **Hook** - Problem statement (30s)
2. **Solution** - Pipeline overview (30s)
3. **Features** - Agent + Human value (1m)
4. **Architecture** - Tech stack (1m)
5. **Live Demo** - CLI showcase (2m)
6. **Web Platform** - UI preview (1m)
7. **Roadmap** - Future plans (30s)
8. **Closing** - CTA (30s)

Total: ~7 minutes
