# Contributing to AI-CAST

Thank you for your interest in contributing to AI-CAST! 🎙️

## How to Contribute

### Reporting Bugs
- Use the [Bug Report template](https://github.com/DaviRain-Su/ai-cast/issues/new?template=bug_report.md)
- Include steps to reproduce
- Include your environment details

### Suggesting Features
- Use the [Feature Request template](https://github.com/DaviRain-Su/ai-cast/issues/new?template=feature_request.md)
- Describe the use case
- Explain why it would be useful

### Pull Requests
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### Code Style
- Follow the existing code style
- Write clear commit messages
- Add comments for complex logic
- Update documentation if needed

## Development Setup

```bash
# Clone the repo
git clone https://github.com/DaviRain-Su/ai-cast.git
cd ai-cast

# Install dependencies
npm install

# Run locally
npm run dev
```

## Questions?

Feel free to open an issue for any questions!
