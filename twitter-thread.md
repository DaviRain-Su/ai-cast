## Tweet 7/12 - CLI Demo
⌨️ CLI in Action

```bash
# Install
npx ai-cast-cli install

# Setup
npx ai-cast-cli init

# Create profile
npx ai-cast-cli profile create \
  --name "AI Daily" \
  --category tech

# Generate podcast
npx ai-cast-cli fetch \
  -u https://example.com/article
npx ai-cast-cli script -i articles.json
npx ai-cast-cli speak -i script.txt
npx ai-cast-cli publish -a podcast.wav
```

8 commands. All JSON-compatible.
